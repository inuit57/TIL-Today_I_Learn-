# 개요 
- JSP 파일이나 CSS, JS 파일 등을 수정해서 서버에 올렸음에도 바로 적용이 되지 않을 경우 시도해볼 수 있는 방법에 대해서 서술한다. 


## 방법 (클라이언트) 
- 브라우저에서 개발자 도구를 F12 버튼을 눌러서 활성화 시킨다. 
- 이후 주소창 왼쪽의 새로고침 버튼을 우클릭하게 되면 "캐시 비우기 및 강력 새로고침"이라는 것이 활성화된다. 
- 그 버튼을 눌러서 캐시도 비우고 강력 새로고침을 진행한다.
